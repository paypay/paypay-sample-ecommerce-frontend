# PayPay Sample Web application


[![License](https://img.shields.io/:license-apache2.0-red.svg)](https://opensource.org/licenses/Apache-2.0)[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fpaypay%2Fpaypay-sample-ecommerce.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fpaypay%2Fpaypay-sample-ecommerce?ref=badge_shield)


A simple and functional E-Commerce site developed in VueJS to demonstrate PayPay's different language SDK's implementations.

Key features:
  - VueJS with Vuex and route
  - Typescript class based components
  - CRUD operations on the cart
  - Polling functionality on order status check

# Client Installation
Install the dependencies and devDependencies and start the server.

```sh
$ cd client
$ yarn
$ yarn serve
```
You should now have the dev server running on http://localhost:8080

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
